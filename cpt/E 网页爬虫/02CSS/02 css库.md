# Less
https://lesscss.org
https://github.com/less/less.js
中文手册 https://www.runoob.com/manual/lessguide/
less诞生于2009年,受sass影响创建的一个开源项目.是一种动态样式语言,为提高CSS应用的灵活性和效率,赋予了动态语言的特性,如变量,继承,运算,函数.less既可以在客户端上运行,也可以在nodejs或rhino平台上运行.

```js
//nodejs安装
npm install -g less
//浏览器环境引入
<link rel="stylesheet/less" type="text/css" href="styles.less" />
<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/3.11.1/less.min.js"></script>
//vscode Easy LESS插件 会把less文件自动生成css文件,html引入的时候,还是要引入css文件

```

```less
//单行注释,编译后不会出现在css文件上
/*多行注释,编译后会出现在文件上*/



//变量
@color: #999;
@bgColor: skyblue;
@width: 200px;
@height: @width * 0.5;
#box {
  display: block;
  color: @color;
  background-color: @bgColor;
  width: @width;
  height: @height;
}





```
