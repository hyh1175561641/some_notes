# 端口被占用的解决方法

```bat
netstat -ano | findstr 8080
最后一项是PID
tasklist | findstr 11416 -kill
```

# hosts文件的修改方式

hosts属性右键->安全->组或用户名->Administrators或Users权限都打开

```bash
# 添加语句
# 用浏览器打开www.aaa.com:8080会映射到127.0.0.1:8080
127.0.0.1 demo.com

# 下面的写法无效，不能连续跳转
127.0.0.1 192.168.0.103
192.168.0.103 dd.com
```
